Build declarative and composable sections, trees, tables, dashboards, and interactive Elm-style TUIs. Easily create new primitives (no component-library limitations).

A port of https://github.com/mattlianje/layoutz to Elm

### Differences if you're coming from `layoutz-hs`

Instead of using `StyleReverse` to modify the background color, there's `Layoutz.withBackgroundColor`.
